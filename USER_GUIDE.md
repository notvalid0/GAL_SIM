# GAL-SIM 用户使用指南

## 系统要求

### Windows 系统要求
- Windows 10 或更高版本
- Python 3.8 或更高版本
- 至少 2GB 可用磁盘空间

## 安装和运行

### 1. 安装 Python
在使用 GAL-SIM 之前，您需要先安装 Python：

1. 访问 [Python 官方网站](https://www.python.org/downloads/)
2. 下载并安装 Python 3.8 或更高版本
3. **重要：安装时请勾选 "Add Python to PATH" 选项**

### 2. 安装 GAL-SIM
1. 运行 `Setup.exe` 安装程序
2. 按照安装向导完成安装

### 3. 配置 API 密钥
1. 打开安装目录，找到 `.env` 文件
2. 使用文本编辑器打开 `.env` 文件
3. 将 `your_api_key_here` 替换为您的 LLM API 密钥
   ```
   LLM_API_KEY=sk-您的密钥
   ```

## 常见问题

### 问题1：应用启动后显示白屏
**可能原因及解决方案：**
1. **Python 未安装或未添加到 PATH**
   - 确保已安装 Python 3.8+
   - 重新安装 Python 时勾选 "Add Python to PATH"
2. **端口 8000 被占用**
   - 检查是否有其他程序占用了 8000 端口
3. **缺少 Python 依赖包**
   - 手动安装依赖：在安装目录下运行 `pip install -r requirements.txt`

### 问题2：无法连接到 LLM 服务
**可能原因及解决方案：**
1. **API 密钥未配置或配置错误**
   - 检查 `.env` 文件中的 `LLM_API_KEY` 设置
2. **网络连接问题**
   - 检查网络连接
   - 检查防火墙设置是否阻止了应用访问网络

### 问题3：应用启动缓慢或无响应
**可能原因及解决方案：**
1. **首次启动需要加载模型**
   - 首次启动可能需要一些时间，请耐心等待
2. **系统资源不足**
   - 确保有足够的内存和 CPU 资源

## 依赖说明

GAL-SIM 需要以下依赖才能正常运行：

- **Python 3.8+**：应用后端运行环境
- **Python 包依赖**：安装程序会自动安装，但需要网络连接
- **网络连接**：用于连接 LLM 服务

## 故障排除

如果遇到问题，请按以下步骤操作：

1. 确认已正确安装 Python 3.8+
2. 检查 `.env` 文件配置是否正确
3. 确认端口 8000 未被其他程序占用
4. 以管理员身份运行应用
5. 检查防火墙和杀毒软件设置
